Extract and persist key insights from the current conversation using Basic Memory tools.

## Task

1. Gather all relevant key insights, decisions, and action items from the conversation.
2. Search Basic Memory for existing notes on related topics using semantic search.
3. If relevant notes exist, update them with new insights; otherwise create new notes.
4. Determine an appropriate folder structure (e.g., "insights", "decisions", "meetings", "projects").
5. Write the notes using clear, descriptive titles that capture the main topic.
6. Include context like date, related topics, and action items where relevant.
7. Store notes under the project that matches your present working directory.

## Output Format

Return a summary of what was captured:
- Number of notes created/updated
- List of note titles and their locations
- Any links to related notes found
